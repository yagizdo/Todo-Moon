## Run Locally

Clone the project

```bash
  git clone https://github.com/yagizdo/Todo-Moon
```

Go to the project directory

```bash
  cd Todo-Moon
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
